los modulos significa que puedes mandar llamar archivos JS desde rl html o desde el propio JavaScript

Hay que acostumbrarse a poner type module en la llamada de archivos JS

No module es para que no se cargue 2 veces el codigo

Para exportar usar la palabra reservada export, pero debes poner esta palabra al inicio de cada linea de codigo que quieras exportar

Solo se puede tener una exportacion default

No se puede hacer export default en una variable, toca primero declarar la variable y despues hacer el export default a el nombre de la constante, esto debido a la elevacion de JS
