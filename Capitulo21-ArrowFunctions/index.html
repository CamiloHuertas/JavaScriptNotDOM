Las arrows functions vinieron de ECMAScript 6 para hacer las funciones mas claras

Nueva forma de declarar funciones anonimas expresadas

Si la arrow function solo va a reproducir una linea de codigo, podemos omitir las llaves

Si la arrow function va a tener solo 1 parametro podemos omitir los parentesis, si va a tener 2 o mas parametros se deben usar los parentesis

Hay que tener cuidado con los objetos ya que las function normales le dicen al codigo donde se encuentra cada metodo del objeto, encambio las arrow functions dicen directamente donde esta alojado el objeto entero, no sus metodos.
