Los protitipos son la base de la programacion orientada a objetos (POO)

dentro de la POO hay 4 conceptos importantes

  • Clases
  • Objetos
      Los objetos tienen 2 conceptos
    • Atributos
    • Metodos

Las clases son modelos a seguir, son algo que sirven como esquema para de ahi basarnos y generar instancias

Un objeto es una instancia de una clase

Un atributo es una caracteristica o propiedad del objeto (Son las variables dentro de un objeto)

Los metodos son las acciones que un objeto puede realizar (Son funciones dentro de un objeto)

cuando estemos trabajando con funciones constructoras en JS tanto los atributos como los metodos deben tener el objeto this.

New nos deja crear una nueva instancia de cualquier tipo, puede incluir una funcion constructora
