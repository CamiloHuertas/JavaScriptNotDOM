Caracteristicas de JavaScript

  • Lenguaje de alto nivel (No nos debemos preocupar por los recursos)
  • Interpretado (Se compila en tiempo de ejecuccion)
  • Dinamico
  • Debilmente tipado (No se debe especificar su naturaleza, esta se reconoce teniendo en base el flujo de la pagina)
  • Multi paradigma (Se pueden usar varios tipos de programaciones)
  • Sensible a MAYUSCULAS y minusculas (variables PERRO y perro son distintas)
  • No necesita los puntos y comas al final de cada linea

Estructura de codigo

  • Los identificadores deben comenzar con:
    • Una letra o
    • Un signo de dolar $
    • Un guion bajo _
    • Nunca con numeros o caracteres especiales
  • Usa Snake_case en:
    • Archivos JS:

      • mi_archivo_javascript.js

    • Usa UPPER_CASE en:
      • Constantes:

        • const UNA_CONSTANTE = 'Soy una constante', PI = 3.1415

    • Usa UpperCamelCase en:
      • Clases:

        • class SerHumano {
        codigo }

    • Usa lowerCamelCase en:
      • Objetos:

        • const unObjeto = {
        objeto }

      • Funciones:

        • function holaMundo {codigo}

      • Instancias:

        • const ajax = new XMLHttpRequest()
